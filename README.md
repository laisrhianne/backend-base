# REST API Model

## Technologies used
- TypeScript
- Eslint and Prettier
- Express
- Typeorm
- Dotenv

## How to Use
- Create a `.env` file based on the `.env.example` file
- Add the token Secret and the Database info in the `.env` file
- Run `yarn` to add the dependencies
- Run `yarn dev` to run the application on `http://localhost:3333`
